## **功能介紹**
### 飼主功能
* 寵物名片管理
* 評論與評分店家
* 條件過濾寵物旅店，包括包括價格、服務、時間
* 生成訂單的同時產生寵物資訊卡，告知店家寵物詳細需求
### 業者功能
* 上架旅店
* 房型管理
* 訂單管理
* 查看評價

### 必要功能
飼主方以及業者方的登入、註冊、修改密碼、忘記密碼
